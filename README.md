# Tic Tac Toe

A simple two-player Tic Tac Toe game built as a single HTML file — no dependencies, no build step.

## Play

Open `tictactoe.html` in any browser.

## Features

- Two-player local multiplayer (X vs O)
- Score tracking across rounds (wins + draws)
- Winning cells highlighted
- Restart button to reset the board

## Tech

Plain HTML, CSS, and JavaScript — no frameworks or libraries.
